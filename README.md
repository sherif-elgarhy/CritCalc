# CritCalc — ICU & Critical Care Calculator

> Fully offline · No ads · No accounts · No internet required

A professional clinical calculator built for ICU physicians, intensivists, emergency medicine doctors, and critical care nurses.

---

## Features

- **57 clinical formulas** across 10 categories
- **Acid-Base Interpreter** — full stepwise analysis with compensation
- **Ventilator Module** — setup, mechanics, gas exchange, weaning readiness
- **ICU Drug Infusions** — 19 drugs with pump rate ↔ dose reverse calculation
- **Vasopressors** — norepinephrine, epinephrine, vasopressin, dopamine, dobutamine, phenylephrine, milrinone
- **Sedation & Analgesia** — propofol, midazolam, dexmedetomidine, fentanyl, morphine, ketamine
- **Electrolytes & Renal** — AG, corrected AG, osmolar gap, CrCl, eGFR, free water deficit
- **Haematology, Obstetrics, Fluids & Burns**
- Calculation history
- Unit toggle (kg/lb, cm/in, mg/dL → mmol/L)
- Scientific calculator

## Stack

- Vanilla JS + Bootstrap 5
- Capacitor 6 (Android wrapper)
- PWA-ready (service worker, manifest)
- Zero runtime dependencies — works 100% offline

## Development Setup

```bash
# Prerequisites
# - Node.js 20+
# - Java 17+
# - Android Studio (for local builds)

npm install
npx cap add android      # first time only
npx cap copy android     # after any www/ changes
npx cap open android     # open in Android Studio
```

## Automated Build (GitHub Actions)

Every push to `main` triggers a build. Download the signed AAB from the Actions tab → Artifacts.

**Required GitHub Secrets:**
| Secret | Description |
|---|---|
| `SIGNING_KEY` | Base64-encoded `.jks` keystore |
| `KEY_ALIAS` | Keystore alias |
| `KEY_STORE_PASSWORD` | Keystore password |
| `KEY_PASSWORD` | Key password |

**Generate keystore:**
```bash
keytool -genkey -v \
  -keystore critcalc-release.jks \
  -keyalg RSA -keysize 2048 \
  -validity 10000 \
  -alias critcalc

# Encode for GitHub secret:
base64 -w 0 critcalc-release.jks
```
